Mugger Mansion
Introduction
The main task for this assignment was to create a game (in C++ with SDL2), which implements a game mechanic and other additional features - for a higher grade.

Mugger Mansion is a C++ / SDL2 game where the player gets trapped inside a Mugger-filled mansion and has to escape. The game allows the player to move around, shoot, pick up health and ammo, and kill muggers.

Features
Player Input, Pickups, AABB Collision, Particle Systems, A* pathfinding, Line of sight calculations, Enemy AI & configurable patrols, Configurable levels (in text files), and Game saving and loading.
Role: Developer
For: University
Grade: 92 / 100 (A2)
Date: January 2017
Type: Top-down shooter
