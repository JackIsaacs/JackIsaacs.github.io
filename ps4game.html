PS4 Game (WIP)
Introduction
As a work in progress the general gameplay for this project is not set in stone. I am planning on making this game a side-scrolling beat em up with a Japanese influenced style. The player will be able to fight enemy ninjas and has to last as long as possible without dying.

As it stands the game includes most core features such as rendering, sound, animation and particle systems. The framework I have created allows me to create a Unity style Entity-Component system which allows me to add components (such as a sprite, playercontroller) to an object!

The camera control and tracking for the game is done in a dedicated Camera class for easy access. The renderer will lookup the in-use Camera and ask for it's translation and zoom matrices to apply to the final transform matrix. The dedicated class also has allowed me to add functions such as screen shake.

Features
Image Rendering, Gravity, Platform Collision, AABB Collision, Sound, Spritesheet Animation, Controller Input, Screen Shake Scripted Levels, Scripted Prefabs, Multithreading (Sound, Level loading), PS4 Keyboard Input, PS4 Message Dialogs.
