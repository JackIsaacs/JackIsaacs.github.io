Pitfall
Introduction
This is a DirectX 11 University project designed to test our skills in 3D maths, matrices and the like. All while creating an experience for the player.

The brief instructed us to add features to our game, along with additional features for more marks, these included 3D direct sound, particles, and quick-switching inputs between keyboard and controller.

Features
Basic physics, AABB collision detection, directional and point lighting, multiple light sources, skyboxes, level loading, non player entities.
