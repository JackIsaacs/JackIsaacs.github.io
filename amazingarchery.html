Amazing Archery
Introduction
Amazing Archery is a game made to specification of an assignment given by my University. We were tasked to create a Android game in Unity which should run on a Nexus 5. The game should revolve around tilt-controlled aiming and gesture control in order to simulate an archery-style game.

Features
Tilt aiming, dragging on the screen to set arrow force, 2 different ranges, dynamic targets, high score saving, high score boards, and wind.
