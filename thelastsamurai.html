The Last Samurai
Introduction
This is a PS4 implementation of a Japanese themed fighting game which features a lone Samurai warrior fighting off hordes of enemy Ninjas trying to take your life.

The game includes basic features such as image rendering, platform collision, sound and animation. The framework for the game is developed in C++ around a Unity-style entity component system.

Features
Image Rendering, Gravity, Platform Collision, AABB Collision, Sound, Spritesheet Animation, Controller Input, Screen Shake, Power ups, Scripted Levels, Scripted Prefabs, Multithreading (Sound, Level loading), PS4 Keyboard Input, PS4 Message Dialogs.
